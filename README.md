# Porta Inversa 🌀
**Computer Graphics Lab 10 Project**

A mind-bending 3D maze game built from scratch using **Java** and raw **OpenGL (JOGL)**. This project demonstrates foundational computer graphics concepts including 3D projection, first-person camera math (LookAt matrices), and geometric transformations.

### Current Features (Phase 1 MVP)
* **JOGL/NEWT Integration:** High-performance, crash-free OpenGL windowing.
* **First-Person Camera:** 3D perspective projection and LookAt vector implementation.
* **3D Geometry:** Basic environment rendering with depth-testing (z-buffer).
